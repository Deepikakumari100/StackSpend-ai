# 🚀 StackSpend AI

> AI-powered SaaS spend optimization platform that helps businesses analyze AI tool expenses, discover savings opportunities, and generate personalized audit reports instantly.

---

# 🌟 Features

## 🔍 AI Spend Audit

Analyze AI tool usage and identify unnecessary subscriptions.

## 💰 Smart Cost Recommendations

Get intelligent recommendations with estimated yearly savings.

## 📄 Personalized PDF Reports

Generate dynamic audit reports based on each user's input.

## 📧 Email Automation

Automatically send audit reports directly to users using EmailJS.

## ⚡ Modern Interactive UI

Smooth animations and responsive design using React + Tailwind CSS + Framer Motion.

## ☁️ Full Stack Deployment Ready

Frontend and backend fully structured for Render deployment.

---

# 🛠️ Tech Stack

## Frontend

* Tailwind CSS

## Backend

## Services & Integrations

* EmailJS

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash id="nhl2o6"
git clone https://github.com/Deepikakumari100/StackSpend-ai.git
```

---

## 2️⃣ Install Frontend Dependencies

```bash id="0x14r6"
cd frontend
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash id="7l7p1w"
cd ../backend
npm install
```

---

# ▶️ Run Project

## Start Frontend

```bash id="6x6x8z"
cd frontend
npm run dev
```

Frontend runs on:

```bash id="9t22cc"
http://localhost:5173
```

---

## Start Backend

```bash id="0c9m3g"
cd backend
npm start
```

Backend runs on:

```bash id="5t4z6j"
http://localhost:5000
```

# 🔐 Environment Variables

Create a `.env` file inside the frontend folder:

```env id="o67m7t"

VITE_EMAILJS_TEMPLATE_ID=your_template_id
```
